Keliauk kartu

A tourism website about Lithuania.

Technologies
	•	HTML5 – page structure
	•	SCSS/SASS – styling
	•	JavaScript – interactivity (slider)
	•	Google Maps API – maps on the contact page

Project Structure

```
Keliauk kartu/
├── index.html          # Главная страница
├── about.html          # Страница "О нас"
├── contact.html        # Страница контактов
├── assets/             # Изображения и иконки
│   ├── link-icons/     # Иконки социальных сетей и контактов
│   └── ...             # Другие изображения
├── styles/             # Стили
│   ├── main.scss       # Основной файл стилей
│   ├── _variables.scss  # Переменные и миксины
│   └── main.css        # Скомпилированный CSS
└── scripts/            # JavaScript
    └── main.js         # Основной скрипт (слайдер)
```

Pages

Home Page (index.html)
	•	Hero section with a call to action
	•	Inspirational travel quote
	•	Interactive slider showcasing attractions
	•	Statistics (routes, team, parks, sites)
	•	Grid of recommended places to visit

About Us (about.html)
	•	Information about the portal
	•	Description of the project’s mission and goals

Contact Page (contact.html)
	•	Contact information (phone, email, address)
	•	Company details
	•	Contact form
	•	Interactive Google Map

Design Features
	•	Responsive design for all devices (mobile, tablet, desktop)
	•	Interactive elements with hover effects
